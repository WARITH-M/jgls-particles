### `Usage`

**index.html**

新建canvas标签，引入particles.js文件即可使用

****

```html
<canvas id="canvas"></canvas>

<script src="particles.js"></script>
```

**VUE**

Packages install

```bash
npm install js-particles
```

新建vue文件，template中新建canvas标签，mounted中引入particles.js

```js
require('particles.js')
```
